## 📂 Estrutura dos Projetos

### 🔹 Credit Score Analysis

Projeto completo de preparação de dados para Machine Learning, com pipeline detalhado e documentação no código.

**Etapas abordadas:**
- 🧹 Tratamento de dados (valores faltantes, outliers, padronização)  
- 📊 Análise exploratória (univariada e bivariada)  
- 🔄 One-Hot Encoding  
- ✂️ Separação treino/teste  
- ⚖️ Balanceamento com SMOTE  
- 📏 Escalonamento com StandardScaler  
- 💾 Exportação das bases tratadas  

💡 As bases geradas neste projeto são utilizadas em modelos de Machine Learning em outro repositório:

- Naive Bayes  
- Decision Tree  

🔗 https://github.com/Ema028/machine-learning

---

### 🔹 E-commerce Data Pipeline

Pipeline de dados simulando fluxo real de negócio.

**Etapas do projeto:**
- 🔗 Integração de dados com **LEFT JOIN** (clientes + transações)  
- 🧹 Tratamento de valores faltantes  
- ✅ Verificação de consistência  
- 📊 Criação de dashboard no Power BI  

💡 Este projeto simula um fluxo real de dados em ambiente de negócio, desde a preparação até a visualização.

---

### 🔹 Churn Analysis

Projeto focado em análise exploratória e geração de insights sobre evasão de clientes.

**Destaques:**
- 🧹 Limpeza e tratamento de dados  
- 📊 Análise exploratória  
- 💡 Insights documentados diretamente no código  

💡 Foco em interpretação de dados e entendimento do comportamento do cliente.
